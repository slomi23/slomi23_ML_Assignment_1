# House Price Prediction
გადმოგვეცემა მონაცემები 1500მდე სახლის აიოვას შტატის ქალაქ ეიმსში. ასევე გვაქვს 79 თვისება/feature რომლებსაც თეორიულად გავლენა უნდა ჰქონდეს სახლების ფასზე. უნდა ვიწინასწარმეტყველოთ ამ მონაცემებზე დაყრდნობით სახლების ფასები. მონაცემები პირველ რიგში შეგროვებულ იქნა განათლების ფუნქციით, მთლიანი კონტესტიც გათვლილია მათთვის ვინც ახლა დგამს პირველ ნაბიჯებს მანქანურ სწავლებაში.

ჩემი მიდგომა იყო რომ  მეცადა რაც შეიძლება მეტი ვარქიტექტურა და პრეპროცესინგის სხვადასხვა გზები რათა დავკვკირვებოდი თუ რომელ ვარინატში იქნებოდა საუკეთესო შედეგი. განვიხილე სამი ვარიანტი: 
1. წრფივი რეგრესია target encoding ით და RFE ით
2. წრფივი რეგრესია target encoding ით და RFE ით ოღონდ kfold ის გამოყენებით ქროსვალიდაციაში
3. წრფივი რეგრესია one hot encoding ით და RFE ით, ასევე kfold ით
4. decision tree ასევე  kfold ით
RFE ის ხშირი გამოყენების მიზეზი იყო ის, რომ საკმაოდ პატარა მონაცემები გვაქვს

ამიტომაც ეს null ები შევავსე ახალი კატეგორიებით. შემდგომ დავდროფე ისეთი სვეტები სადაც მოდა იყო  null.

ჰიპერპარამეტრებს რაც შეეხება, ტესტირებების შედეგად აღმოჩდნა რომ rfe ის საშUალებით 15 სვეტის დატოვება იყო ოპტიმალური. თუმცა OHE ს ვარიანტში 8 აღმოჩნდა საუკეთესო ვარიანტი.
